# Security Policy

## Supported Versions

| Version | Supported |
| --- | --- |
| 1.0.x | Yes |
| < 1.0.0 | No |

## Reporting a Vulnerability

We take the security of Rotordyn.ai seriously. If you find any vulnerabilities, please do not disclose them publicly. Instead:
1. Email details of the vulnerability to `security@rotordyn.ai`.
2. Include steps to reproduce, impact details, and mitigation suggestions.
3. We will acknowledge receipt of the report within 48 hours and work on a fix.
